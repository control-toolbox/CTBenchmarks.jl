Core CPU Benchmarks

Ubuntu Latest

This benchmark suite evaluates optimal control problems on a standard CPU platform using GitHub Actions runners.

Note

The linear solver is MUMPS for all experiments.

  • Models: adnlp, exa, jump

  • Tolerance: 1.0e-8
